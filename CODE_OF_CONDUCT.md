# Code of Conduct

## Our Pledge
We pledge to make participation in our community a harassment-free experience.

## Our Standards
Examples of behavior that contributes to a positive environment include:
- Being respectful
- Accepting constructive criticism

## Enforcement Responsibilities
Project maintainers are responsible for enforcing this code.

## Scope
This Code of Conduct applies within project spaces.

## Enforcement
Instances of abusive behavior may be reported.

## Enforcement Guidelines
Project maintainers will review complaints fairly.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
